개인정보처리방침

최종 업데이트: 2024년 6월 26일

안전한 비밀번호 생성기는 사용자의 개인정보 보호를 최우선으로 합니다.

1. 수집하는 개인정보

본 앱은 다음과 같은 개인정보를 수집합니다:

  • 생성된 비밀번호: 사용자가 생성하고 저장한 비밀번호
  • 라벨 정보: 비밀번호에 사용자가 입력한 라벨
  • 생성 시간: 비밀번호가 생성된 시간

2. 개인정보 저장 및 처리

모든 개인정보는 다음과 같이 처리됩니다:

  • 로컬 저장: 모든 데이터는 사용자의 기기에만 저장됩니다
  • 외부 전송 없음: 인터넷을 통한 데이터 전송이 없습니다
  • 암호화: 저장된 데이터는 기기의 기본 보안 시스템으로 보호됩니다

3. 개인정보 사용 목적

수집된 개인정보는 다음 목적으로만 사용됩니다:

  • 비밀번호 생성 및 저장 기능 제공
  • 저장된 비밀번호 관리 및 편집
  • 앱 기능 개선 및 사용자 경험 향상

4. 개인정보 공유

본 앱은 다음과 같은 경우를 제외하고 개인정보를 제3자와 공유하지 않습니다:

  • 사용자의 명시적 동의가 있는 경우
  • 법적 요구사항에 따른 경우

5. 개인정보 보호

개인정보 보호를 위해 다음과 같은 조치를 취합니다:

  • 모든 데이터는 사용자 기기에만 저장
  • 외부 서버로의 데이터 전송 없음
  • 기기의 기본 보안 시스템 활용

6. 개인정보 삭제

사용자는 언제든지 다음 방법으로 개인정보를 삭제할 수 있습니다:

  • 앱 내에서 개별 비밀번호 삭제
  • 앱 삭제 시 모든 데이터 자동 삭제

7. 쿠키 및 추적 기술

본 앱은 쿠키나 추적 기술을 사용하지 않습니다.

8. 아동의 개인정보

본 앱은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 개인정보처리방침 변경

본 개인정보처리방침은 필요에 따라 변경될 수 있으며, 변경 시 앱 내에서 공지합니다.
